# MVP Spec

## Goal

Build the smallest version of the assignment that still feels like a real AI-first email client.

## Must Have

- responsive PWA email client
- unified inbox
- account switcher for Gmail, Office 365, Yahoo IMAP, and AOL IMAP
- email detail view
- compose
- reply
- forward
- search
- labels
- archive
- delete
- AI summary
- AI reply draft
- AI priority
- short `CLAUDE.md`
- architecture doc
- workflow doc
- agent list
- automated tests
- Vercel-ready project

## Nice To Have

- security risk badge
- follow-up recommendation
- keyboard-friendly search
- polished empty states

## Do Not Build

- contacts
- tasks
- notes
- calendar
- full CRM
- complex backend before the frontend MVP is complete

## Success Criteria

The reviewer should be able to open the live app and understand within one minute:

- this is an email client
- it supports multiple providers conceptually
- the inbox is unified
- AI helps summarize and respond
- the codebase has agent workflow discipline
- the project is tested and documented
